# Kraftreaktor Lenzburg/Aarau × DynoForce

Gemeinsames Branding-Set für die DynoForce Event-Plattform mit offiziellen Motiven aus beiden Kraftreaktor-Standorten.

| Feld | Datei | Format |
|---|---|---|
| Header Banner | `header-banner-2400x600.jpg` | 2400 × 600 px, 4:1; Lenzburg und Aarau mit Kraftreaktor-Logo links |
| Eventlogo | `eventlogo-df-1000x1000.png` | 1000 × 1000 px, reines DF-Logo |
| Hallenlogo | leer | wird nicht angezeigt |
| Sponsor Footer | `sponsor-footer-2500x500.jpg` | 2500 × 500 px, Kraftreaktor und DF |

## Einstellungen

- Primärfarbe: `#1D1D1D`
- Header-Grösse: `100 %`
- Eventlogo: `100 %`
- Hallenlogo anzeigen: deaktiviert
- Sponsor-Grösse: `100 %`

## Quellen

- Logo und Lenzburg-Motiv: `https://kraftreaktor.ch`
- Aarau-Motiv: `https://kraftreaktor-aarau.ch`
- Header-Komposition: OpenAI-Bildbearbeitung, anschließend pixelgenaue Einbindung des offiziellen SVG-Logos
- Reproduzierbar über `create-branding.cjs`

Die Vorlage wird im Demo-Account der Event-Plattform unter **Kraftreaktor Lenzburg/Aarau** gespeichert.
