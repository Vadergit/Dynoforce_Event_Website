# GrindelBoulder Bassersdorf × DynoForce

Branding-Set für die DynoForce Event-Plattform auf Basis des offiziellen Auftritts von GrindelBoulder.

| Feld | Datei | Format |
|---|---|---|
| Header Banner | `header-banner-2400x600.jpg` | 2400 × 600 px, 4:1; offizielle Kletterszene mit GB-Logo links |
| Eventlogo | `eventlogo-df-1000x1000.png` | 1000 × 1000 px, reines DF-Logo |
| Hallenlogo | leer | wird nicht angezeigt |
| Sponsor Footer | `sponsor-footer-2500x500.jpg` | 2500 × 500 px, GrindelBoulder und DF |

## Einstellungen

- Primärfarbe: `#0ABE8F`
- Header-Grösse: `100 %`
- Eventlogo: `100 %`
- Hallenlogo anzeigen: deaktiviert
- Sponsor-Grösse: `100 %`

## Quellen

- Logo und Kletterfoto: `https://grindelboulder.ch`
- Header-Komposition: OpenAI-Bildbearbeitung, anschließend pixelgenaue Einbindung des offiziellen SVG-Logos
- Reproduzierbar über `create-branding.cjs`

Die Vorlage wird im Demo-Account der Event-Plattform unter **GrindelBoulder Bassersdorf** gespeichert.
