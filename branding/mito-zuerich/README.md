# MITO Bouldering Zürich × DynoForce

Branding-Set für die DynoForce Event-Plattform auf Basis des offiziellen Auftritts von MITO Bouldering Zürich.

| Feld | Datei | Format |
|---|---|---|
| Header Banner | `header-banner-2400x600.jpg` | 2400 × 600 px, 4:1; offizielles Hallenfoto mit MITO-Logo links |
| Eventlogo | `eventlogo-df-1000x1000.png` | 1000 × 1000 px, reines DF-Logo |
| Hallenlogo | leer | wird nicht angezeigt |
| Sponsor Footer | `sponsor-footer-2500x500.jpg` | 2500 × 500 px, MITO und DF |

## Einstellungen

- Primärfarbe: `#7D1413`
- Header-Grösse: `100 %`
- Eventlogo: `100 %`
- Hallenlogo anzeigen: deaktiviert
- Sponsor-Grösse: `100 %`

## Quellen

- Logo und Hallenfotos: `https://www.mito-bouldering.ch`
- Header-Komposition: OpenAI-Bildbearbeitung, anschließend pixelgenaue Einbindung des offiziellen SVG-Logos
- Reproduzierbar über `create-branding.cjs`

Die Vorlage wird im Demo-Account der Event-Plattform unter **MITO Bouldering Zürich** gespeichert.
