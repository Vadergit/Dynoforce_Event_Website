# BOUBA Boulder Baden × DynoForce

Branding-Set für die DynoForce Event-Plattform auf Basis des offiziellen Auftritts von BOUBA Boulder Baden.

| Feld | Datei | Format |
|---|---|---|
| Header Banner | `header-banner-2400x600.jpg` | 2400 × 600 px, 4:1; offizielle Hallenaufnahme mit BOUBA-Logo links |
| Eventlogo | `eventlogo-df-1000x1000.png` | 1000 × 1000 px, reines DF-Logo |
| Hallenlogo | leer | wird nicht angezeigt |
| Sponsor Footer | `sponsor-footer-2500x500.jpg` | 2500 × 500 px, BOUBA und DF |

## Einstellungen

- Primärfarbe: `#345F6B`
- Header-Grösse: `100 %`
- Eventlogo: `100 %`
- Hallenlogo anzeigen: deaktiviert
- Sponsor-Grösse: `100 %`

## Quellen

- Logo und Hallenfoto: `https://boulder-baden.ch`
- Header-Komposition: OpenAI-Bildbearbeitung, anschließend pixelgenaue Einbindung des offiziellen Logos
- Reproduzierbar über `create-branding.cjs`

Die Vorlage wird im Demo-Account der Event-Plattform unter **BOUBA Boulder Baden** gespeichert.
